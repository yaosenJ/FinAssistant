
# 板块数据说明

## 数据源概况

| 数据类型 | 数据源 | 备注 |
|----------|--------|------|
| 板块历史行情（日线） | 同花顺 (THS) | 东财已封，新浪无历史数据，THS 是唯一可用源 |
| 板块成分股 | 新浪财经 (Sina) | 东财已封，THS 无成分股 API 且反爬封锁分页 |

## 数据文件清单

| 文件 | 数据源 | 内容 | 数量 |
|------|--------|------|------|
| `industry_sectors.json` | THS | 行业板块每日行情（2026-01-01~07-10） | 90个板块，每个124条日线 |
| `industry_cons.json` | 新浪 | 行业板块成分股 | 49个板块，2431只股票 |
| `concept_sectors.json` | THS | 概念板块每日行情（2026-01-01~07-10） | 374个板块，每个124条日线 |
| `concept_cons.json` | 新浪 | 概念板块成分股 | 213个板块，8126只股票 |

## 字段说明

### 行情数据（*_sectors.json）

| 字段 | 含义 | 示例 |
|------|------|------|
| industry_name / concept_name | 板块名称 | "半导体"、"AI PC" |
| industry_ts_code / concept_ts_code | 板块代码 | "885909"、"309121" |
| trade_date | 交易日期 | "2026-01-02" |
| open | 开盘点位 | 2562.82 |
| close | 收盘点位 | 2595.08 |
| high | 最高点位 | 2595.08 |
| low | 最低点位 | 2429.36 |
| pct_chg | 涨跌幅(%) | 1.25（由收盘价计算） |
| change_data | 涨跌额 | 32.26（由收盘价计算） |
| vol | 成交量 | 12345678 |
| amount | 成交额（万元） | 8523.45 |
| pct_change | 振幅(%) | 6.45（由最高最低价计算） |
| turnover_rate | 换手率(%) | null（THS不提供） |

**计算公式：**
- `pct_chg` = (当日close - 前一日close) / 前一日close * 100
- `change_data` = 当日close - 前一日close
- `pct_change` = (当日high - 当日low) / 前一日close * 100（振幅）

### 成分股数据（*_cons.json）

| 字段 | 含义 |
|------|------|
| name | 板块名称 |
| node | 新浪板块节点ID |
| stock_count | 成分股数量 |
| stocks[].code | 股票代码 |
| stocks[].name | 股票名称 |
| stocks[].market | 市场（SH/SZ） |
| stocks[].trade | 最新价 |
| stocks[].changepercent | 涨跌幅(%) |
| stocks[].mktcap | 总市值（万） |
| stocks[].nmc | 流通市值（万） |
| stocks[].pe | 市盈率 |
| stocks[].pb | 市净率 |

## 数据源不一致说明

THS 概念板块（374个）与新浪概念板块（213个）名称不一致，仅约66个重叠。行业板块方面，THS（90个）和新浪（49个）命名规则不同，新浪是更细粒度的行业分类。

两套数据分开存储，使用时需注意：
- 按板块名称关联行情数据和成分股数据
- 部分板块只有行情数据无成分股，反之亦然

## API 封禁情况

| 数据源 | 状态 | 说明 |
|--------|------|------|
| 东方财富 (Eastmoney) | 全部封禁 | `push2.eastmoney.com` 返回 RemoteDisconnected |
| 同花顺 (THS) 行情 | 可用 | `stock_board_*_index_ths` 正常 |
| 同花顺 (THS) 成分股 | IP被封 | 频繁抓取触发反爬，IP被Nginx封禁（403 Forbidden） |
| 新浪财经 (Sina) 成分股 | 可用 | `Market_Center.getHQNodes` + `getHQNodeData` 接口可用 |

## 爬虫脚本

| 脚本 | 用途 | 用法 |
|------|------|------|
| `crawl_sector_data.py` | 爬取板块每日行情（THS） | `python crawl_sector_data.py [industry\|concept\|all]` |
| `crawl_sector_cons.py` | 爬取板块成分股（新浪财经API） | `python crawl_sector_cons.py [industry\|concept\|all]` |
| `crawl_sector_cons_ths.py` | 爬取板块成分股（THS网页，DrissionPage） | `python crawl_sector_cons_ths.py [industry\|concept\|all]` |
| `crawl_financial_reports.py` | 爬取三大财务报表（新浪） | `python crawl_financial_reports.py [sh\|sz\|all]` |
