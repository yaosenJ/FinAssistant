# 板块轮动分析技能

## 触发条件

用户询问板块行情、板块排名、资金流向、板块对比、轮动趋势时触发。典型问题：

**板块排名查询**：
- "最近一周涨幅最大的行业板块有哪些？"
- "今天成交额最大的概念板块是哪些？"
- "哪些板块连续3天上涨？"
- "哪些板块出现了跌停潮？"

**资金流向查询**：
- "半导体板块最近一个月资金是在流入还是流出？"
- "电力板块的成交额趋势如何？"
- "哪些板块资金在持续流入？"

**板块对比查询**：
- "白酒和新能源板块最近谁涨得多？"
- "AI概念和机器人概念的走势对比"
- "银行、证券、保险最近20天谁强谁弱？"

**轮动趋势查询**：
- "最近市场热点在哪里？"
- "哪些板块在加速上涨？哪些在退潮？"
- "当前热门板块和冷门板块分别有哪些？"

**板块深度分析**：
- "白酒板块今天涨跌家数如何？"
- "半导体板块的成分股涨跌分布"
- "AI概念和机器人概念的成分股重叠度有多高？"

## 依赖工具组

| 工具组 | 包含工具 | 功能 |
|--------|----------|------|
| `sector-ranking` | get_sector_ranking, get_sector_top_gainers, get_sector_top_losers, get_sector_summary | 板块排名、连续涨跌板块、市场概览 |
| `sector-rotation` | get_sector_momentum, get_sector_rotation, get_sector_strength, get_hot_cold_sectors | 动量分析、轮动识别、强度排名、冷热分类 |
| `sector-compare` | compare_sectors, compare_sector_trend | 多板块涨跌对比、趋势强度对比 |
| `sector-detail` | get_constituent_distribution, get_sector_money_flow, get_sector_correlation | 成分股分布、资金流向、板块关联度 |

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `sector_type` | str | 否 | 板块类型：industry(行业) / concept(概念)，默认 industry |
| `sector_name` | str | 视情况 | 板块名称，支持模糊匹配，如"白酒"、"半导体" |
| `days` | int | 否 | 统计天数，默认根据问题自动判断 |
| `top_n` | int | 否 | 返回数量，默认 10-15 |

### 复合问题处理

对于复合问题（如"最近热点板块有哪些？资金流向如何？"），按以下顺序调用：

1. 先调用 `get_hot_cold_sectors` 或 `get_sector_ranking` 获取热点板块
2. 再调用 `get_sector_money_flow` 分析资金流向
3. 最后调用 `get_sector_momentum` 判断动量趋势

## 分析框架

### 板块强弱判断

| 维度 | 强势信号 | 弱势信号 |
|------|---------|---------|
| 涨幅 | 连续多日上涨，累计涨幅靠前 | 连续多日下跌，累计跌幅靠前 |
| 资金 | 成交额持续放大，资金流入 | 成交额持续萎缩，资金流出 |
| 动量 | 短期涨幅 > 中期涨幅（加速） | 短期涨幅 < 中期涨幅（减速） |
| 成分股 | 涨跌比 > 2:1，多只涨停 | 涨跌比 < 1:2，多只跌停 |

### 板块轮动识别

| 阶段 | 特征 | 操作建议 |
|------|------|---------|
| 启动期 | 短期涨幅突增，成交额放大，动量评分转正 | 关注，可小仓位参与 |
| 加速期 | 连续上涨，资金持续流入，动量评分高位 | 持有，注意止盈 |
| 退潮期 | 短期涨幅回落，资金流入放缓，动量评分下降 | 减仓，切换方向 |
| 冰点期 | 持续下跌，资金大量流出 | 回避，等待转机 |

## 注意事项

1. **板块类型区分**：行业板块(90个)和概念板块(374个)数据分开存储，需明确类型
2. **模糊匹配**：板块名称支持模糊匹配，如"半导体"可匹配"半导体及元件"
3. **数据时效**：板块数据更新到最新交易日，非交易日显示上一交易日数据
4. **成分股差异**：不同数据源的成分股可能不完全一致，以同花顺数据为准
5. **免责声明**：所有分析仅供参考，不构成投资建议
